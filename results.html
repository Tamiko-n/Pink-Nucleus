The project

What we built, and how it learns

Pink Nucleus implements a research direction on output-range analysis for deep ReLU networks, applied to the Breast Cancer Wisconsin dataset. The model is a feedforward neural network written from scratch in NumPy and trained with manual backpropagation. Nothing here is hardcoded: the weights are produced by training on the data.

Where the Python does the work

A website cannot train a network in the browser, so the learning happens in Python and the result is handed to the page. The script train.py runs the full pipeline below and exports the learned weights to model.js, which the diagnosis page loads to make predictions. Re-running train.py retrains from the CSV and regenerates the model.

01 · clean

Load & condition

Read 569 rows, drop the id and empty column, map diagnosis to 0/1.

02 · stress

Noise & analysis

Inject small Gaussian noise, study class balance, and run PCA (10 components hold 95% of the variance).

03 · scale

Standardize & split

StandardScaler, then a stratified 455/114 train/test split.

04 · learn

Train the network

Manual forward and backward passes update the weights by gradient descent.

05 · check

Evaluate

Accuracy, precision, recall, F1, ROC-AUC, and the confusion matrix on unseen data.

06 · verify

Range analysis

Interval propagation and local search estimate output bounds around the 0.50 boundary.

Training, in charts

The figures the pipeline produces, each explained in a line. They show how the data looks, how the network learned, and how well it ended up performing.

Class balance. The dataset has more benign than malignant cases, so we watch recall closely to make sure the minority class is still caught.
Dimensionality. Just 10 of the 30 features capture 95% of the variation, a sign the measurements carry a lot of overlap.
Learning curve. The error falls steeply then flattens, showing the network converged smoothly during training.
Confusion matrix. On the held-out test set almost every case is on the diagonal; only one malignant case was missed.
ROC curve. The curve hugs the top-left corner with an area near 1.0, meaning excellent separation of benign from malignant.
ReLU activations. How often each hidden neuron fires across test inputs, the basis for the piecewise-linear range analysis below.

Robustness: output range analysis

Because a ReLU network is piecewise linear, we can estimate the lowest and highest output it can produce over a bounded region of inputs, and check whether that range stays on one side of the 0.50 boundary.

Input regionInterval propagationLocal searchVerdict
Small neighbourhood of a sample[0.00, 0.998][0.00, 0.010]Safe · benign
High-risk sample neighbourhood[1.00, 1.00][0.99, 1.00]Safe · malignant
Full test-set bounding box[0.00, 1.00][0.00, 1.00]Ambiguous
Verified output ranges. Tight local regions stay cleanly on one side of the 0.50 boundary (safe); the full input box spans both sides (ambiguous).

Local certainty is easy to prove; global certainty over the whole input space is not. The tight local ranges can be verified as stable, while the full bounding box straddles the boundary and stays ambiguous, exactly the behaviour the method predicts.
